# Node.js configuration

## Installing packages

#### 1. Npm

```
npm init
npm install
npm install -g nodemon

```

#### 2. Express

```
npm install express --save
```

#### 3. EJS

```
npm install ejs --save
```

#### 4. MySQL

```
 npm install mysql --save
```

#### 5. Consign

```
npm install consign --save
```
